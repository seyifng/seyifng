## 🚀 Featured Projects

### 👗 DripFinder — Cloud-Backed 3D Closet Simulator
> A full-stack platform to manage and visualize digital clothing assets in an interactive 3D virtual closet.

- Led frontend development in **React + TypeScript**, building the primary UI for asset management and closet customization
- Created custom **3D clothing meshes in Blender** using Python scripts, integrated with Spring Boot APIs
- Optimized delivery through **AWS S3** for smooth previews and interactive customization

`React` `TypeScript` `Spring Boot` `AWS S3` `Blender (Python)`

---

### 🦷 Smiles With Joy Dental — Production SPA
> A live, client-facing dental practice website built for real patient engagement.

- Designed and deployed a **mobile-first, responsive** single-page application using React, Vite, and TypeScript
- Implemented **SEO best practices** and integrated Google Reviews API with a custom reviews UI
- Managed full deployment pipeline with Git and Vercel

`React` `TypeScript` `Vite` `Vercel` `Google APIs`

---

### 🔁 SkillSwap — Community Skill Exchange Platform *(Hackathon)*
> A peer-to-peer platform for trading skills — built in 24 hours.

- Built core **React + TypeScript** components for skill listings and user interactions from scratch
- Designed a **badge-based reputation system** and structured relational database tables in Supabase

`React` `TypeScript` `Supabase`

---

### 📅 BookEase — Appointment Booking System

`Next.js` `Flask` `MySQL`
